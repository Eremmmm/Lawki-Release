Features

Credential Management

Store logins with title, username, password, URL, and notes. Organize with custom categories — pick a name, color, and icon. Filter, search, switch between list and grid views, and set expiration dates per entry.

  • Custom categories with name, color & icon
  • Expiration dates per credential
  • List & grid view toggle
  • Fast search & filter

Password Generator

Generate strong, unique passwords with full control over the rules. Instantly evaluate strength with a visual indicator based on NIST SP 800-63B recommendations — Weak, Medium, or Strong.

  • Custom length & character rules
  • Uppercase, numbers, symbols
  • Strength meter (NIST SP 800-63B)

Security Dashboard

Get a real-time security score (0–100%) computed from the strength of all your passwords. Spot reused credentials, track statistics, and monitor recent activity — all at a glance.

  • Global security score (0–100%)
  • Duplicate password detection
  • Strong vs. weak credential stats
  • Recent activity feed

Sync

Move your vault.
Without the cloud.

Lawki lets you transfer your data directly between devices — peer to peer, with no intermediary server.

Wi-Fi (Local Network)

Pair two devices on the same network with a QR code or a 6-character code. Automatic device discovery — no manual IP configuration needed.

USB Cable

Connect via USB cable for a direct transfer. Lawki auto-detects the tethered connection and walks you through setup with a guided assistant.

Flexible Import Modes

Choose how to handle existing credentials: replace everything, skip duplicates, or manually pick which entries to import — full control, always.

Browser Extension

Autofill,
natively.

Lawki integrates directly with your browser through a native messaging host — no remote server in the loop. Credentials are matched by domain, subdomain, or reverse match, and injected securely.

Firefox Chrome
  • Domain & subdomain matching
  • Secure native messaging protocol
  • Pre-shared token authentication
